Computer Case

Concept design for a case that a sleek and seamless design with easy access to internals.

This is a concept for a case that was created for practice over a weekend. I wanted to create a custom case that was different from many ofthe more agressive designs on the market, and came up with this as a result. The main body is built in 3 peices, with the right side connected to the front of the case. It is designed to slide out like a drawer to provide easy access to the internals without the need for screws. Made with Solidworks and rendered with Solidworks visualize and HDR Lightroom.

Nutrifier

A pacifier designed to provide nutrients to malnourished infants

The Nutrifier was designed for Nutrivide, a company I am working on within the Honors College Innovation Lab at Rutgers. This is the second revision of the design, which focuses on visualizing most of the important mechanical elements so that features can be more easily explained during product pitches. Made using Solidworks and rendered with Solidworks Visualize

Chandelier

A lighting fixture concept made using 3DS Max

This chandelier was created as a practice project, and was my first model made using 3DS Max. It is a design that I had in mind for a while, but would have been harder to accomplish in solidworks due to its organic form. I used this as an opporitunity to teach myself the basics of a new program so that I could start to visualize some of my more freeform designs.

Nutrifier v 1.0

The first concept of the Nutrifier, created for marketing purposes.

This is the first iteration of the Nutrifier, created before I started designing the internal mechanism. This was also my first attempt at creating an original model in solidworks, so it was a learning process. This model was used so that we had something to help other visualize the pacifier when we pitched the idea, but did not take into account the mechanism that would be created later. This resulted in it having a very different visual design from the current model.
